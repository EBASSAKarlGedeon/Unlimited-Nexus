# Terminology of the CAS Era

## Name for CAS Users

Individuals capable of interacting with the Cognitive-Actuation System are officially called **Actuants**.

The term comes from _cognitive actuation_, referring to people able to **translate knowledge into physical phenomena**.

Subtypes exist depending on specialization:

- **Analysts** – high knowledge capacity but limited execution
    
- **Operators** – strong actuation capability
    
- **Manufacturers** – specialized in creating stable objects or structures
    
- **Strategists** – Actuants capable of coordinating multiple abilities
    

In casual language, people simply say **Acts**.

# Global Network

## Inter-Noetic Network (INN)

The modern internet evolved into the **Inter-Noetic Network**.

_Noetic_ refers to knowledge and cognition.

The network was redesigned to support:

- rapid scientific information exchange
    
- real-time cognitive data sharing
    
- CAS research databases
    
- neural model libraries used by Actuants
    

For many Actuants, the Inter-Noetic Network functions almost like an **external cognitive extension**, allowing them to access structured knowledge frameworks instantly.

# CAS Manufacturing

## Artifacts

**Artifacts** are physical objects created through **Manufacturing-type actuation**.

Unlike temporary phenomena, these creations are **stable and persistent**, meaning the CAS process permanently restructures matter.

Examples include:

- CAS conductive materials
    
- neural resonance devices
    
- actuation amplifiers
    
- structural materials with unusual properties
    

Artifacts are extremely valuable because they allow **non-Actuants to benefit from CAS technologies**.

Governments strictly regulate high-grade Artifacts.


---

## SAS — Brain Communication Layer

The meaning of SAS refers to the **neural communication protocol** that enables this process.

Through a specialized neural interface, brain signals are mapped and translated into **transferable actuation patterns**.

This allows multiple Actuants to function as a **coordinated combat network**.

---

# Brain Map Interface

The **Brain Map Interface (BMI)** is the central control system used by SAS operators.

The interface visualizes a user's **cognitive architecture**

Factors influencing this include:

- cognitive stability
    
- neural processing speed
    
- knowledge compatibility
    
- mental endurance
    

However, excessive synchronization risks **neural overload**, memory corruption, or identity fragmentation.
